# nFraction

## English Summary
The problem requires writing a program to solve a system of linear equations with $n$ unknowns. Given the coefficient matrix and the solution matrix of the equations, the program needs to output the values of the unknown variables. The recommended approach is to use Gaussian elimination method to solve the system of equations.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک برنامه برای حل یک سیستم معادلات خطی با $n$ مجهول نوشت. با داشتن ماتریس ضرایب و ماتریس جواب معادلات، برنامه باید مقادیر متغیرهای ناشناخته را خروجی دهد. روش توصیه شده برای حل این معادلات استفاده از روش حذف گاوسی می‌باشد.
