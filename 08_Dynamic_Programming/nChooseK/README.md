# nChooseK

## English Summary
The problem involves calculating the number of ways to choose $k$ objects from $n$ distinct objects using a recursive formula. The task is to implement a recursive function to compute this value efficiently.

## Persian Summary / خلاصه فارسی
در این مسئله باید تعداد راه‌های انتخاب $k$ شیء از $n$ شیء متمایز را با استفاده از یک فرمول بازگشتی محاسبه کرد. وظیفه شما پیاده‌سازی یک تابع بازگشتی برای محاسبه این مقدار به صورت بهینه است.
