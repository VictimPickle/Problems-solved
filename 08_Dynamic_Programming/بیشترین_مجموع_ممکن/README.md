# بیشترین_مجموع_ممکن

## English Summary
Given an array of integers, find the subarray with the largest sum among all possible subarrays and output the sum of its elements.

## Persian Summary / خلاصه فارسی
در این مسئله، یک آرایه از اعداد داده شده است و باید زیرآرایه‌ای با بزرگترین مجموع بین تمام زیرآرایه‌های ممکن را پیدا کرده و مجموع اعضای آن را خروجی دهید.
