# digitAdd

## English Summary
The problem involves repeatedly summing the digits of a given natural number $n$ until $n$ becomes a single-digit number, and then outputting that final single-digit number.

## Persian Summary / خلاصه فارسی
در این مسئله باید به صورت مکرر اعداد یک عدد طبیعی $n$ را جمع کرده تا زمانی که $n$ به یک عدد تک‌رقمی تبدیل شود و سپس آن عدد تک‌رقمی را خروجی دهیم.
