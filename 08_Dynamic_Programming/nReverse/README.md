# nReverse

## English Summary
The problem requires writing a program that, using only one recursive function, prints a sequence based on a given number $n$ in a specific pattern. The sequence starts from $n$ and goes down to 1, then back up to $n$.

## Persian Summary / خلاصه فارسی
این مسئله نیازمند نوشتن یک برنامه است که با استفاده از یک تابع بازگشتی، دنباله‌ای بر اساس یک عدد $n$ داده شده در یک الگوی خاص چاپ می‌کند. دنباله از $n$ شروع شده و به 1 می‌رسد، سپس دوباره به $n$ بالا می‌رود.
