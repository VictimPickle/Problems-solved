# چوپان_دلسوز

## English Summary
The problem involves a shepherd named Cyrus-Ali who needs to transport his sheep and wolves across a river using a limited-capacity boat. If at any point the number of wolves exceeds the number of sheep on either side of the river or in the boat, the wolves will eat the sheep. The goal is to determine the minimum number of trips Cyrus-Ali needs to make to safely transport all animals without any losses.

## Persian Summary / خلاصه فارسی
در این مسئله، چوپانی به نام سیروس‌علی باید گوسفندان و گرگ‌های خود را با استفاده از یک قایق با ظرفیت محدود از یک سوی رودخانه به سوی دیگر منتقل کند. اگر تعداد گرگ‌ها در هر طرف رودخانه یا درون قایق بیشتر از تعداد گوسفندان شود، گرگ‌ها گوسفندان را می‌خورند. هدف این است که حداقل تعداد سفرهایی که سیروس‌علی باید انجام دهد تا تمام حیوانات را بدون هیچ تلفاتی به سوی دیگر منتقل کند، محاسبه شود.
