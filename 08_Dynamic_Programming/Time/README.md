# Time

## English Summary
Given the initial time displayed on a clock before the battery runs out and the number of seconds elapsed since then, the task is to determine the current time on the clock after the specified duration.

## Persian Summary / خلاصه فارسی
با داشتن زمان اولیه نمایش داده شده روی یک ساعت قبل از تمام شدن باتری و تعداد ثانیه‌های گذشته از آن زمان، وظیفه تعیین زمان فعلی روی ساعت پس از مدت زمان مشخص شده است.
