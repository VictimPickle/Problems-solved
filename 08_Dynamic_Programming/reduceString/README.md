# reduceString

## English Summary
The problem involves simplifying a given string by removing consecutive identical characters until no two consecutive characters are the same. This process is done recursively. The task is to implement a recursive function to simplify the string.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک رشته داده شده را ساده‌سازی کرده و حروف متوالی یکسان را حذف کرد تا هیچ دو حرف متوالی یکسانی باقی نماند. این فرایند به صورت بازگشتی انجام می‌شود. وظیفه شما پیاده‌سازی یک تابع بازگشتی برای ساده‌سازی رشته است.
