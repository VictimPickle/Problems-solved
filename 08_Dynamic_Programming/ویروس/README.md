# ویروس

## English Summary
The problem involves creating a virus that disrupts the execution of a program in three different ways based on specific substrings. The goal is to find the shortest combination of substrings that can cause the program to encounter all three issues: runtime error, memory limit, and time limit. The input consists of three strings, and the output should be the length of the desired string that includes all three specified substrings.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک ویروس ایجاد کنید که اجرای برنامه را با توجه به سه زیر رشته خاص، به سه روش مختلف اختلال دهد. هدف این است که کوتاهترین ترکیبی از زیر رشته ها را پیدا کنید که باعث مواجهه برنامه با سه مشکل مشخص شده شود: خطای زمان اجرا، محدودیت حافظه و محدودیت زمان. ورودی شامل سه رشته است و خروجی باید طول رشته مورد نظر که شامل همه سه زیر رشته مشخص شده است، باشد.
