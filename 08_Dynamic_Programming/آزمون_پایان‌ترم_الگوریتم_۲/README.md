# آزمون_پایان‌ترم_الگوریتم_۲

## English Summary
Given a set of $n$ questions with corresponding scores, the task is to find the number of contiguous intervals where the sum of scores equals a target value $x$. The goal is to determine the count of such intervals.

## Persian Summary / خلاصه فارسی
در این مسئله، باید برای مجموعه‌ای از $n$ سوال با امتیازهای مخصوص، تعداد بازه‌های متوالی را پیدا کنید که مجموع امتیازات آن‌ها برابر با یک مقدار هدف $x$ باشد. هدف این است که تعداد این بازه‌ها را محاسبه کنید.
