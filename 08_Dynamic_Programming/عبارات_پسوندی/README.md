# عبارات_پسوندی

## English Summary
Given a postfix expression representing a mathematical expression with only the operators *, +, /, and -, the task is to find the original expression and evaluate its value.

## Persian Summary / خلاصه فارسی
با داشتن یک عبارت پسوندی که یک عبارت ریاضی با عملگرهای *، +، / و - را نمایش می‌دهد، وظیفه‌ی شما این است که عبارت اصلی را پیدا کرده و مقدار آن را محاسبه کنید.
