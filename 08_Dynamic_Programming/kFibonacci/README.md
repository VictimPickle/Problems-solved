# kFibonacci

## English Summary
The problem involves calculating the $n$-th term of a $k$-Fibonacci sequence using a recursive function. The $k$-Fibonacci sequence is defined by a recursive formula where each term is the sum of the previous $k$ terms, with specific base cases. The task is to compute the $n$-th term of the sequence given $n$ and $k$.

## Persian Summary / خلاصه فارسی
در این مسئله باید جمله $n$-ام دنباله $k$-فیبوناچی را با استفاده از یک تابع بازگشتی محاسبه کنیم. دنباله $k$-فیبوناچی توسط یک فرمول بازگشتی تعریف شده است که هر جمله آن مجموع $k$ جمله قبلی آن است، با شرایط اولیه خاص. وظیفه این است که با دادن $n$ و $k$، جمله $n$-ام دنباله را محاسبه کنید.
