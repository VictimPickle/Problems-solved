# baseOperation

## English Summary
The program reads a base $b$ and two natural numbers $n$ and $m$. It converts the numbers to base $b$, performs addition, subtraction, multiplication, and division operations on them in base $b$, stores the results in separate arrays, and finally prints them.

## Persian Summary / خلاصه فارسی
این برنامه یک پایه $b$ و دو عدد طبیعی $n$ و $m$ می‌خواند. اعداد را به پایه $b$ تبدیل کرده، عملیات جمع، تفریق، ضرب و تقسیم را بر روی آن‌ها در پایه $b$ انجام داده، نتایج را در آرایه‌های جداگانه ذخیره کرده و در نهایت چاپ می‌کند.
