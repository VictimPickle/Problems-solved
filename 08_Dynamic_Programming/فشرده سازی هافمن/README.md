# فشرده سازی هافمن

## English Summary
The problem involves implementing the Huffman coding algorithm for data compression. Given a text and its corresponding Huffman encoding, the task is to decode the text using the Huffman tree constructed from the given Huffman encoding.

## Persian Summary / خلاصه فارسی
این مسئله شامل پیاده‌سازی الگوریتم کدگذاری هافمن برای فشرده‌سازی داده‌ها می‌شود. با داشتن یک متن و کدگذاری هافمن متناظر آن، وظیفهٔ ما این است که متن را با استفاده از درخت هافمن ساخته شده از کدگذاری هافمن داده شده، رمزگشایی کنیم.
