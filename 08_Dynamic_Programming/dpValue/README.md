# dpValue

## English Summary
Given $n$ coins with values $\lbrace v_{0},v_{1},...,v_{n-1} \rbrace$ and a target sum $s$, the task is to write a dynamic programming algorithm to find the minimum number of coins needed to make up the sum $s$.

## Persian Summary / خلاصه فارسی
با داشتن $n$ سکه با ارزش‌های $\lbrace v_{0},v_{1},...,v_{n-1} \rbrace$ و یک مقدار هدف $s$، وظیفه این است که یک الگوریتم برنامه‌نویسی پویا بنویسید تا حداقل تعداد سکه‌های مورد نیاز برای ساختن مجموع $s$ را پیدا کند.
