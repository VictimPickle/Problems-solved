# تو_آزادی_که_بخوری!

## English Summary
The problem involves helping Master Shifu prepare the maximum number of dumplings possible given a limited amount of raw materials and a magical powder that can transform into any raw material. The goal is to maximize the number of dumplings cooked based on the available raw materials and the magical powder.

## Persian Summary / خلاصه فارسی
در این مسئله باید به استاد شیفو کمک کرد تا با استفاده از مواد اولیه محدود و پودر جادویی، بیشترین تعداد دامپلینگ ممکن را بپزد. هدف اصلی این است که تعداد دامپلینگ‌های پخته شده بر اساس مواد اولیه موجود و پودر جادویی را بیشینه کنیم.
