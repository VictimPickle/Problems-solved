# جنگ_تن_به_تن

## English Summary
General Pokuio, facing defeat in the war, convinces Commander Gogorio to agree to a one-on-one combat for the remaining soldiers. He needs a program to sort the soldiers based on the length of their names using pointers and dynamic memory allocation, ensuring stable sorting for equal-length names.

## Persian Summary / خلاصه فارسی
ژنرال پوکویو، در مواجهه با شکست در جنگ، فرمانده گوگوریو را متقاعد می‌کند تا به یک نبرد یک به یک برای سربازان باقی‌مانده موافقت کند. او نیاز دارد که یک برنامه بنویسد که سربازان را بر اساس طول نامشان با استفاده از پوینترها و تخصیص حافظه پویا مرتب کند، تضمین کننده مرتب سازی پایدار برای نام‌های با طول یکسان.
