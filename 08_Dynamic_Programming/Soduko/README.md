# Soduko

## English Summary
The problem involves solving a partially filled Sudoku puzzle of size $n \times n$. The rules of Sudoku require each row, column, and $\sqrt{n} \times \sqrt{n}$ subgrid to contain all numbers from 1 to $n$. The task is to complete the Sudoku puzzle based on the given input.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک پازل سودوکو $n \times n$ نیمه حل شده را حل کرده و تکمیل کنید. قوانین سودوکو از شما می‌خواهد که هر سطر، ستون و زیرجدول $\sqrt{n} \times \sqrt{n}$ شامل تمامی اعداد از ۱ تا $n$ باشد. وظیفه شما تکمیل کردن پازل سودوکو بر اساس ورودی داده شده است.
