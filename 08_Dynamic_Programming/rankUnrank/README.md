# rankUnrank

## English Summary
The problem involves writing a program to rank and unrank $k$ subsets. Given an integer $n$, the program should either rank or unrank a subset based on the input. If the input is 1, a subset from 1 to $n$ is provided, and if the input is 2, the rank of a subset is given. The task is to handle ranking and unranking of subsets efficiently.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک برنامه برای رتبه‌بندی و رتبه‌گذاری $k$ زیرمجموعه بنویسید. با دادن یک عدد صحیح $n$، برنامه باید یک زیرمجموعه را بر اساس ورودی رتبه‌بندی یا رتبه‌گذاری کند. اگر ورودی 1 باشد، یک زیرمجموعه از 1 تا $n$ داده می‌شود و اگر ورودی 2 باشد، رتبه یک زیرمجموعه داده می‌شود. وظیفه این است که بازیابی و رتبه‌بندی زیرمجموعه‌ها را به صورت کارآمد انجام دهد.
