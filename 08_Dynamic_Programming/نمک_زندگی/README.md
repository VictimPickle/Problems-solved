# نمک_زندگی

## English Summary
The problem involves determining whether a person named Liteh can respond to messages from friends within specified time intervals. Liteh can chat with a limited number of friends simultaneously. The task is to decide if Liteh can reply to all friends within their respective waiting times.

## Persian Summary / خلاصه فارسی
در این مسئله باید تعیین کرد که آیا فردی به نام لیته می‌تواند در زمان‌های مشخصی به پیام‌های دوستان خود پاسخ دهد یا خیر. لیته می‌تواند با تعداد محدودی از دوستان همزمان چت کند. وظیفه این است که تصمیم گرفته شود که آیا لیته می‌تواند به همه دوستان در زمان‌های انتظار مشخصی که دارند پاسخ دهد یا خیر.
