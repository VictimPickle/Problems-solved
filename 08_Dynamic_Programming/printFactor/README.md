# printFactor

## English Summary
The problem requires writing a recursive program that takes an input number $n$ and prints all prime factorizations of $n$ where the sum of the factors equals $n$. The output should display the factorizations in lexicographic order.

## Persian Summary / خلاصه فارسی
این مسئله نیازمند نوشتن یک برنامه بازگشتی است که یک عدد $n$ را از ورودی گرفته و تمامی تجزیه‌های اولیه $n$ را چاپ می‌کند که مجموع این اعداد برابر با $n$ باشد. خروجی باید تجزیه‌ها را به ترتیب **lexicographic** نمایش دهد.
