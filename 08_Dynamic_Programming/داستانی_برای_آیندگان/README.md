# داستانی_برای_آیندگان

## English Summary
The problem presents a simplified version of a program containing three types of commands: `assign`, `cout`, and `goto`. The task is to simulate the program execution line by line and output the result if the program terminates, or -1 if it does not. The program manipulates variables and performs arithmetic operations, with the output being the values printed by the `cout` commands. The final output should be the remainders of the outputs modulo $10^9+7$.

## Persian Summary / خلاصه فارسی
این مسئله نسخه‌ی ساده‌شده‌ای از یک برنامه با سه نوع دستور `assign`، `cout` و `goto` را ارائه می‌دهد. وظیفه شما این است که اجرای برنامه را خط به خط شبیه‌سازی کرده و نتیجه را چاپ کنید اگر برنامه پایان یابد و در غیر این صورت -1 را چاپ کنید. برنامه متغیرها را تغییر می‌دهد و عملیات حسابی را انجام می‌دهد، و خروجی نهایی باید باقی‌مانده‌های خروجی‌ها را بر $10^9+7$ چاپ کند.
