# نمره_۲۰

## English Summary
Given a message of length $n$ consisting of lowercase English letters, the task is to encrypt the message by moving the last letter to the beginning and shifting each letter to the next one in the alphabet. This encryption process is repeated $k$ times. The goal is to find the final encrypted message after $k$ iterations.

## Persian Summary / خلاصه فارسی
در این مسئله، یک پیام به طول $n$ که شامل حروف کوچک انگلیسی است داده شده است. وظیفه این است که پیام را با جابه‌جایی حرف آخر به ابتدا و تبدیل هر حرف به حرف بعدی در الفبا رمزگذاری کنید. این فرآیند رمزگذاری $k$ بار تکرار می‌شود. هدف این است که پس از $k$ مرحله، پیام رمزگذاری شده نهایی را بیابید.
