# Battery

## English Summary
Given the charging behavior of a tax collector's mobile phone, where it takes $i+1$ minutes to go from $i$ charge units to $i+1$ charge units, the task is to determine the time required for the phone to charge from 0 to $k$ charge units.

## Persian Summary / خلاصه فارسی
با توجه به رفتار شارژ گوشی تلفن همراه یک مرد مالیاتچی، که برای رسیدن از $i$ واحد شارژ به $i+1$ واحد شارژ، $i+1$ دقیقه زمان می‌برد، وظیفه ما تعیین زمان مورد نیاز برای شارژ گوشی از 0 تا $k$ واحد شارژ است.
