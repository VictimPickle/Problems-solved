# تایپیست_بزرگ

## English Summary
Given a string typed by Pasha, consisting of lowercase English letters and '=' sign, the task is to reconstruct the original string by removing the '=' signs that were mistakenly typed instead of backspace. The characters are to be typed in the correct order.

## Persian Summary / خلاصه فارسی
در این مسئله، باید رشته اصلی که توسط پاشا نوشته شده و در آن علامت '=' به جای backspace استفاده شده، با حذف علامت '=' ها به ترتیب صحیح بازسازی شود.
