# ترکیب_چهارکه

## English Summary
Given $n$ types of coins with different values and a target value $x$, find the number of ways to make the target value $x$ using the coins. The order of coins does not matter.

## Persian Summary / خلاصه فارسی
در این مسئله، با $n$ نوع سکه با ارزش‌های مختلف و یک مقدار هدف $x$، تعداد روش‌های ساختن مقدار هدف $x$ با استفاده از سکه‌ها را بیابید. ترتیب سکه‌ها اهمیتی ندارد.
