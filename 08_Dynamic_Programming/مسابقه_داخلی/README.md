# مسابقه_داخلی

## English Summary
Given an array representing the power of soldiers in an army, the task is to find the length of the longest subarray where the difference between the maximum and minimum power is at most 1. Two pointers and dynamic programming are required to solve this problem efficiently.

## Persian Summary / خلاصه فارسی
در این مسئله با داده شدن یک آرایه که نماینده قدرت سربازان در یک ارتش است، باید طول بلندترین زیرآرایه‌ای که اختلاف بیشینه و کمینه قدرت آن حداکثر ۱ باشد را پیدا کنید. برای حل این مسئله به دقیقاً دو پوینتر و استفاده از داینامیک مموری نیاز است.
