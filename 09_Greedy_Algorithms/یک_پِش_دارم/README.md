# یک_پِش_دارم

## English Summary
The problem involves simulating a game called "پِش" where players take turns saying specific phrases to each other. The goal is to generate the dialogue for a given number of roles and their corresponding names.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک بازی به نام "پِش" را شبیه‌سازی کرده و دیالوگ‌های آن را برای تعداد داده شده از نقش‌ها و نام‌هایشان تولید کنید.
