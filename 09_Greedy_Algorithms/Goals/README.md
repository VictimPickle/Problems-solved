# Goals

## English Summary
The problem involves determining if a given sequence of goals scored in a football match can be split into two halves (first half and second half) such that the goals are scored in chronological order within each half. The input consists of the total number of goals, the duration of the first half, the duration of the second half, and the timestamps of when each goal was scored. The task is to output 'YES' if it is possible to split the goals as described, and 'NO' otherwise.

## Persian Summary / خلاصه فارسی
در این مسئله باید تشخیص داد که آیا یک دنباله از گل‌های زده شده در یک بازی فوتبال را می‌توان به دو نیمه (نیمه‌ی اول و دوم) تقسیم کرد به‌طوری که گل‌ها در هر نیمه به ترتیب زمانی باشند. ورودی شامل تعداد کل گل‌ها، مدت زمان نیمه‌ی اول، مدت زمان نیمه‌ی دوم و زمان زدن هر گل است. وظیفه این است که در صورت امکان تقسیم گل‌ها به شرح داده شده، `YES` و در غیر این صورت `NO` را خروجی دهد.
