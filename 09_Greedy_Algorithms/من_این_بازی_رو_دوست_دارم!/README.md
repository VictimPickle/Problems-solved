# من_این_بازی_رو_دوست_دارم!

## English Summary
Given a game where two players take turns selecting either the first or the last number from a list of numbers on a table to add to their score, the task is to determine the maximum score the first player can achieve assuming both players play optimally.

## Persian Summary / خلاصه فارسی
در این مسئله، با داشتن یک بازی که دو نفر به نوبت اعدادی را از یک لیست انتخاب می‌کنند و به امتیاز خود اضافه می‌کنند، هدف تعیین کردن بیشترین امتیازی است که نفر اول می‌تواند کسب کند با فرض اینکه هر دو نفر به بهترین نحو ممکن بازی می‌کنند.
