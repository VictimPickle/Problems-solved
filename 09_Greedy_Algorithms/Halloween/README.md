# Halloween

## English Summary
The problem involves dividing a given number of chocolates between two students, Asghar and Faizeh, in a specific way where Asghar gets more chocolates than Faizeh. The task is to determine the number of ways the chocolates can be divided between the two students.

## Persian Summary / خلاصه فارسی
در این مسئله باید تعدادی شکلات به دو دانش‌آموز به نام‌های اصغر و فائزه به نحوی تقسیم کرد که اصغر بیشتر از فائزه شکلات بگیرد. وظیفه‌ی شما تعیین تعداد حالات مختلف تقسیم شکلات بین این دو دانش‌آموز است.
