# آزمون_پایان‌ترم_الگوریتم_۳

## English Summary
Given a set of $n$ questions with corresponding scores, the task is to find the number of contiguous intervals of questions such that the sum of scores in each interval is strictly less than a given value $x$. Only one interval can be selected to answer all the questions within that interval.

## Persian Summary / خلاصه فارسی
در این مسئله، باید تعداد بازه‌های متوالی از سوالات را پیدا کنید که مجموع نمرات آن‌ها از یک مقدار مشخص $x$ کمتر باشد. تنها مجاز به انتخاب یک بازه برای پاسخگویی به تمام سوالات در آن بازه هستید.
