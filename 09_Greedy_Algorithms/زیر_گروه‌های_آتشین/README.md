# زیر_گروه‌های_آتشین

## English Summary
Given a group of $n$ furious heroes, not all of them are always available. The task is to find all possible subsets of available heroes. The subsets should be printed in lexicographic order.

## Persian Summary / خلاصه فارسی
در این مسئله، باید برای یک گروه از $n$ قهرمان آتشین که همه آنها همیشه در دسترس نیستند، تمام زیرمجموعه‌های ممکن از قهرمان‌های در دسترس را پیدا کنید. زیرمجموعه‌ها باید به ترتیب لغوی چاپ شوند.
