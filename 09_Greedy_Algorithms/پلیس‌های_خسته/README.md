# پلیس‌های_خسته

## English Summary
Given a map represented by an array of 'T's (thieves) and 'P's (police), where each police can catch only one thief and cannot catch a thief who is more than k units away, find the maximum number of thieves that can be caught by the police.

## Persian Summary / خلاصه فارسی
با داشتن یک نقشه که توسط یک آرایه از 'T'ها (دزدها) و 'P'ها (پلیس‌ها) نمایش داده شده است، جایی که هر پلیس تنها می‌تواند یک دزد را دستگیر کند و نمی‌تواند دزدی را که بیش از k واحد با او فاصله دارد دستگیر کند، بیشترین تعداد دزدهایی که توسط پلیس‌ها می‌توانند دستگیر شوند را پیدا کنید.
