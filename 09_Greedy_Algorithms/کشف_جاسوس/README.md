# کشف_جاسوس

## English Summary
Given an array of sorted integers representing the power of senior soldiers, the task is to find the number of pairs of soldiers that can defeat a spy with a given approximate power. The goal is to select the maximum number of pairs efficiently in linear time complexity using exactly two pointers and dynamic memory.

## Persian Summary / خلاصه فارسی
با داشتن یک آرایه از اعداد صحیح مرتب شده که نمایانگر قدرت سربازان ارشد است، وظیفه این است که تعداد جفت‌های سربازانی که می‌توانند یک جاسوس با قدرت تقریبی داده شده را شکست دهند را پیدا کنید. هدف انتخاب حداکثر تعداد جفت‌ها به صورت کارآمد در پیچیدگی زمانی خطی با استفاده از دقیقاً دو پوینتر و حافظه داینامیک است.
