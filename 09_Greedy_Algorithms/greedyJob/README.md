# greedyJob

## English Summary
Given a set of intervals represented as $(s_i, f_i)$ where $f_i > s_i$, the task is to find a subset $J \subset I$ of non-overlapping intervals with maximum cardinality using a greedy approach.

## Persian Summary / خلاصه فارسی
در این مسئله، با داشتن مجموعه‌ای از بازه‌ها که به صورت $(s_i, f_i)$ نمایش داده شده‌اند و $f_i > s_i$، هدف این است که با استفاده از روش حریصانه، یک زیرمجموعه $J \subset I$ از بازه‌های غیر همپوشان با بیشترین تعداد را پیدا کنیم.
