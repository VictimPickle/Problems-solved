# طول_و_مجموع_ارقام

## English Summary
Given two integers $m$ and $s$, the task is to find the smallest and largest numbers with $m$ digits and a digit sum of $s$. The numbers should be non-negative integers in base 10 and should not start with zero.

## Persian Summary / خلاصه فارسی
با دو عدد صحیح $m$ و $s$، وظیفه‌ی شما این است که کوچکترین و بزرگترین اعداد با $m$ رقم و مجموع ارقام $s$ را پیدا کنید. اعداد باید اعداد صحیح غیر منفی در مبنای ۱۰ باشند و با صفر شروع نشوند.
