# LongComp

## English Summary
Given two numbers represented with an integer value and the number of trailing zeros, the task is to compare these numbers and output '<' if the first number is smaller, '>' if the first number is larger, and '=' if they are equal.

## Persian Summary / خلاصه فارسی
با داشتن دو عدد که با یک مقدار صحیح و تعداد صفرهای چسبانده شده به انتهای آنها نمایش داده شده‌اند، وظیفه مقایسه این دو عدد و چاپ '<' اگر عدد اول کوچکتر باشد، '>' اگر عدد اول بزرگتر باشد و '=' اگر برابر باشند است.
