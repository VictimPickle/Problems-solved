# تی_ای_ها_و_گنج_ابدی

## English Summary
The problem involves finding a specific pattern, resembling the letter "L", in a given grid. The task is to count the number of such patterns in the grid and use this count as the password to unlock a historical lock. The grid consists of '*' and '.' characters, and the 'L' pattern is defined by a sequence of '*' characters forming an L shape. Only one attempt is allowed to enter the correct password.

## Persian Summary / خلاصه فارسی
در این مسئله باید الگوهای خاصی که شبیه حرف "L" در یک جدول داده شده وجود دارند را پیدا کرده و تعداد آن‌ها را بشمارید و این تعداد را به عنوان رمز برای باز کردن یک قفل تاریخی استفاده کنید. جدول از کاراکترهای '*' و '.' تشکیل شده است و الگوی 'L' توسط یک دنباله از کاراکترهای '*' که یک شکل L را تشکیل می‌دهند تعریف می‌شود. تنها یک فرصت برای وارد کردن رمز صحیح وجود دارد.
