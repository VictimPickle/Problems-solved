# اسب

## English Summary
Given a chessboard of size $n\times n$, with the current position of a knight and the destination position provided, the task is to calculate the minimum number of moves required for the knight to reach the destination. Knights move in an L-shape pattern.

## Persian Summary / خلاصه فارسی
در این مسئله، در یک صفحه شطرنج $n\times n$، با داده شدن موقعیت فعلی اسب و موقعیت مقصد، باید حداقل تعداد حرکت‌های لازم برای رسیدن اسب به مقصد را محاسبه کنید. اسب با الگوی حرکتی شکل L حرکت می‌کند.
