# آرامش_درونی_جنگجوی_اژدها

## English Summary
The problem involves solving the Tower of Hanoi puzzle, where the goal is to move a stack of disks from one rod to another while following specific rules. The task is to implement an algorithm to move the disks with preserving their order using only two auxiliary rods.

## Persian Summary / خلاصه فارسی
این مسئله شامل حل معمای برج هانوی است، که هدف آن انتقال یک پشته از بشقاب‌ها از یک چوب به چوب دیگر با رعایت قوانین خاص است. وظیفه شما پیاده‌سازی یک الگوریتم برای انتقال بشقاب‌ها با حفظ ترتیب آنها با استفاده از دو چوب کمکی است.
