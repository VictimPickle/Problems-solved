# سفید_برفی

## English Summary
Given the time intervals at which birds chirp, the task is to determine the time when the k-th chirp is heard by a character named Snow White. The chirps occur periodically for each bird, and Snow White moves when she hears the k-th chirp.

## Persian Summary / خلاصه فارسی
با داشتن فواصل زمانی که پرندگان جیک‌جیک می‌کنند، وظیفه‌ی شما تعیین زمانی است که پرنده‌ی $k$-امین صدای جیک‌جیک را که توسط شخصی به نام سفید برفی شنیده می‌شود، مشخص کنید. جیک‌جیک‌ها به صورت دوره‌ای برای هر پرنده رخ می‌دهند و سفید برفی هنگام شنیدن $k$-امین صدای جیک‌جیک حرکت می‌کند.
