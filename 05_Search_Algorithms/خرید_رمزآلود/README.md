# خرید_رمزآلود

## English Summary
The problem involves determining whether a given pair of strings can be transformed into each other by rearranging their characters. The task is to implement a program with a time complexity of $O(n)$ using only conditions and loops, without using arrays or built-in functions.

## Persian Summary / خلاصه فارسی
در این مسئله باید تشخیص داد که آیا جفت داده شده از رشته‌ها می‌توانند با جابه‌جایی حروف خود به یکدیگر تبدیل شوند یا خیر. وظیفه این است که یک برنامه با پیچیدگی زمانی $O(n)$ با استفاده تنها از شرایط و حلقه‌ها، بدون استفاده از آرایه‌ها یا توابع آماده پیاده‌سازی کنید.
