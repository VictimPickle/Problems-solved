# تی_ای_و_شرطبندی

## English Summary
Given the number of laps, length of each lap, number of cars, their tire and refueling change times, and their speeds, the task is to determine the winning car after all laps based on the total time taken by each car to complete the race.

## Persian Summary / خلاصه فارسی
با داشتن تعداد دورها، طول هر دور، تعداد ماشین‌ها، زمان تعویض لاستیک و سوخت‌گیری آن‌ها، و سرعت آن‌ها، وظیفه تعیین ماشین برنده پس از اتمام تمامی دورها بر اساس زمان کلی که هر ماشین برای اتمام مسابقه نیاز دارد است.
