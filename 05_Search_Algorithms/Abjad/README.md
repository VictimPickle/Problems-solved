# Abjad

## English Summary
Given a numerical value in the Abjad language encoding, the task is to find all words in the Abjad language that have a total value equal to the given number. The words should consist of a maximum of 5 letters from the Abjad alphabet and be printed in lexicographically sorted order.

## Persian Summary / خلاصه فارسی
با داشتن یک مقدار عددی در رمزگذاری زبان ابجد، وظیفه این است که تمام کلماتی را که مجموع ارزش آن‌ها برابر با عدد داده شده است، پیدا کنید. این کلمات باید حاوی حداکثر 5 حرف از الفبای ابجد باشند و به ترتیب مرتب‌سازی لغوی چاپ شوند.
