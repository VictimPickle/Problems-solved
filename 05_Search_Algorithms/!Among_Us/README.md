# !Among_Us

## English Summary
Given a sequence of representatives in a committee meeting, where everyone except one person is solving exercises and one person is a spy, the task is to find the spy among them. The input consists of the number of test cases, the length of the array of representatives, and the representatives themselves. The output should provide the index of the spy in each test case.

## Persian Summary / خلاصه فارسی
در این مسئله باید از بین یک دنباله از نماینده‌ها در یک جلسه کمیته، که همه به جز یک نفر تمرین‌ها را حل می‌کنند و یک نفر جاسوس است، جاسوس را پیدا کنید. ورودی شامل تعداد تست‌ها، طول آرایه نماینده‌ها و خود نماینده‌ها است. خروجی باید شامل اندیس جاسوس در هر تست باشد.
