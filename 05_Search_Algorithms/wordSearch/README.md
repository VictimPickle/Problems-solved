# wordSearch

## English Summary
The problem involves searching for a given string within a matrix of English letters. The search can be performed in vertical and horizontal directions starting from any element in the matrix. If the string is found, a matrix with corresponding numbers indicating the order of letters in the found string is returned. Otherwise, "Not found" is printed.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک رشته داده شده را درون یک ماتریس از حروف انگلیسی جستجو کرده و در صورت پیدا شدن، یک ماتریس با اعداد متناظر با ترتیب حروف رشته در ماتریس برگردانده می‌شود. در غیر اینصورت، عبارت "Not found" چاپ می‌شود.
