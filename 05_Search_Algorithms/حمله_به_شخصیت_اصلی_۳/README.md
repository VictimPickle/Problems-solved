# حمله_به_شخصیت_اصلی_۳

## English Summary
The main character is facing an attack by a swarm of bugs. The character can choose a point on the x-axis and destroy all bugs within a certain range with a limited number of attacks. The goal is to find the minimum integer value of R such that all bugs can be destroyed with a maximum of k attacks.

## Persian Summary / خلاصه فارسی
شخصیت اصلی با حمله‌ی یک گله از سوسک‌ها روبرو است. او می‌تواند یک نقطه روی محور $x$ را انتخاب کرده و با تعداد محدودی حمله، تمام سوسک‌هایی که در فاصله‌ی مشخصی قرار دارند را نابود کند. هدف پیدا کردن کمترین مقدار صحیح $R$ است که با حداکثر $k$ حمله، تمام سوسک‌ها نابود شوند.
