# فامیل_دور!

## English Summary
Given a binary search tree representing a family tree, the task is to find the $p$-th common ancestor of two given nodes in the tree. If the common ancestor exists, the program should output the value of that ancestor; otherwise, it should return -1.

## Persian Summary / خلاصه فارسی
در این مسئله، یک درخت جستجوی دودویی که نماینده‌ی درخت خانوادگی است داده شده است. هدف این است که $p$-امین جد مشترک دو گره داده شده در درخت را پیدا کنید. اگر جد مشترک وجود داشته باشد، برنامه باید مقدار آن جد را خروجی دهد؛ در غیر این صورت باید -1 را برگرداند.
