# nKnight

## English Summary
Given an integer $n$, the task is to find the number of ways to place $\frac{n}{2}$ knights and $\frac{n}{2}$ bishops on an $n \times n$ chessboard such that they do not threaten each other.

## Persian Summary / خلاصه فارسی
با داشتن یک عدد صحیح $n$، وظیفه این است که تعداد راه‌های قرار دادن $\frac{n}{2}$ شاه و $\frac{n}{2}$ فیل بر روی یک تخته شطرنج $n \times n$ بدست آورید به طوری که یکدیگر را تهدید نکنند.
