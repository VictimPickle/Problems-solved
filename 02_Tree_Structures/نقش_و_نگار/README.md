# نقش_و_نگار

## English Summary
The problem involves implementing a painter's model on a square paper grid. The painter starts with a white $n \times n$ grid, colors some squares black, and then repeats the process for white squares in subsequent stages. The input specifies the dimensions of the model and the number of stages, along with the initial grid configuration. The task is to output the final grid configuration after applying the painter's model for the specified number of stages.

## Persian Summary / خلاصه فارسی
این مسئله شامل پیاده‌سازی مدل نقاش روی یک شبکه مربعی کاغذ است. نقاش با یک شبکه سفید $n \times n$ شروع می‌کند، برخی از مربع‌ها را سیاه می‌کند، و سپس این فرآیند را برای مربع‌های سفید در مراحل بعدی تکرار می‌کند. ورودی ابعاد مدل و تعداد مراحل را همراه با تنظیمات اولیه شبکه مشخص می‌کند. وظیفه این است که خروجی نهایی شبکه را پس از اعمال مدل نقاش برای تعداد مشخص شده از مراحل، چاپ کند.
