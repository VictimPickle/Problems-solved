# عدالت_حتی_در_شب_یلدا

## English Summary
Given the inorder and preorder traversals of a binary tree representing inheritance and legacy division among family members, Batman wants to find out how many nodes in the tree have the property that the sum of values in the left subtree is equal to the sum of values in the right subtree. 

## Persian Summary / خلاصه فارسی
با داده شدن پیمایش‌های inorder و preorder یک درخت دودویی که نماینده تقسیم ارث و میراث بین اعضای خانواده است، بتمن می‌خواهد بداند چند راس در این درخت ویژگی دارند که مجموع مقادیر در زیردرخت سمت چپ آنها برابر با مجموع مقادیر در زیردرخت سمت راست آنها است.
