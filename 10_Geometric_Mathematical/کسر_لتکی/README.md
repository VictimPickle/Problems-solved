# کسر_لتکی

## English Summary
The problem involves generating and printing a fraction expression in LaTeX after a certain number of iterations. The fraction expression has a specific pattern of nested fractions that need to be expanded up to the given number of iterations.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک عبارت کسری در $LaTeX$ را بعد از تعداد مشخصی از مراحل چاپ کنیم. این عبارت کسری الگوی خاصی از کسرهای تو در تو را دارد که باید تا تعداد دفعات مشخص شده گسترش یابد.
