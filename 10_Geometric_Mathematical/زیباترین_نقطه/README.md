# زیباترین_نقطه

## English Summary
Given a set of beautiful points on a plane, the task is to find the most beautiful point, defined as the point with the minimum ugliness value, where ugliness for a point A is defined as the sum of distances from A to its two closest beautiful points B and C.

## Persian Summary / خلاصه فارسی
با داشتن مجموعه‌ای از نقاط زیبا در صفحه، هدف پیدا کردن زیباترین نقطه است که به عنوان نقطه‌ای با کمترین میزان زشتی تعریف می‌شود. زشتی برای یک نقطه A به عنوان جمع فواصل از A به دو نقطه زیبایی نزدیکترین B و C تعریف می‌شود.
