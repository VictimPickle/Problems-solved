# عدد_خوب

## English Summary
The problem involves finding the first "good" number that has a specific number of natural divisors. The professor is interested in numbers constructed by summing integers from 1 to n and wants a program to find the first good number with k divisors. The solution should be implemented using only conditions and loops.

## Persian Summary / خلاصه فارسی
در این مسئله باید اولین عدد "خوب" را که تعداد خاصی از مقسوم‌علیه‌های طبیعی دارد پیدا کنید. استاد علاقه‌مند است که اعدادی که از جمع اعداد ۱ تا n ساخته شده‌اند را بررسی کند و می‌خواهد برنامه‌ای برای پیدا کردن اولین عدد خوب با k مقسوم‌علیه طبیعی داشته باشد. راه حل باید تنها با استفاده از شرایط و حلقه‌ها پیاده‌سازی شود.
