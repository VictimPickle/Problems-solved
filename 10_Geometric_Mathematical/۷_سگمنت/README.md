# ۷_سگمنت

## English Summary
Given a valid scientific notation of a natural number, where the number is to be displayed on a digital screen using seven segments, the task is to determine the amount of electricity consumed for displaying the number.

## Persian Summary / خلاصه فارسی
با داشتن یک نمایش علمی معتبر از یک عدد طبیعی، جایی که عدد باید روی یک صفحه‌ی دیجیتال با استفاده از هفت سگمنت نمایش داده شود، وظیفه تعیین میزان برق مصرفی برای نمایش این عدد است.
