# expon

## English Summary
Given input values $p$ and $x$, the problem requires calculating the value of $e^x$ using the provided formula up to the $p$-th term. The output should be rounded to 4 decimal places.

## Persian Summary / خلاصه فارسی
با داشتن مقادیر ورودی $p$ و $x$، مسئله نیازمند محاسبه‌ی مقدار $e^x$ با استفاده از فرمول داده شده تا مرتبه $p$ است. خروجی باید تا چهار رقم اعشار گرد شود.
