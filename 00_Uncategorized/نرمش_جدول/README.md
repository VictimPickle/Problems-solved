# نرمش_جدول

## English Summary
Given a table of size $n \times n$ with each cell colored either black or white, perform three types of operations: rotate the table 90 degrees clockwise, mirror the table vertically, or mirror the table horizontally. After applying a series of these operations, determine the final state of the table.

## Persian Summary / خلاصه فارسی
در این مسئله یک جدول $n \times n$ با رنگ‌های سیاه و سفید داده شده است. سه نوع عمل بر روی جدول انجام داده می‌شود: چرخاندن جدول 90 درجه به صورت عقربه‌های ساعت، قرینه کردن جدول نسبت به خط افقی وسط جدول و قرینه کردن جدول نسبت به خط عمودی وسط جدول. پس از انجام یک سری از این عمل‌ها، وضعیت نهایی جدول را تعیین کنید.
