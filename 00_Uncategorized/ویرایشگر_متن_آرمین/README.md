# ویرایشگر_متن_آرمین

## English Summary
The problem involves analyzing a text editor designed by Armin in the Paradise Island. The editor provides information about the number of words and their lengths in a text. The task is to find the length of the longest word outside parentheses and the number of words inside parentheses.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک ویرایشگر متن طراحی شده توسط آرمین در جزیره پارادیس مورد تجزیه و تحلیل قرار داده شود. این ویرایشگر اطلاعاتی در مورد تعداد کلمات و طول آن‌ها در یک متن فراهم می‌کند. وظیفه این مسئله، پیدا کردن طول بلندترین کلمه خارج از پرانتز و تعداد کلمات درون پرانتزهاست.
