# دونات_عسلی

## English Summary
Given a bee hive structure where each hive is connected to its adjacent hives forming a tree-like structure, the task is to find the shortest distance between two specified hives.

## Persian Summary / خلاصه فارسی
در این مسئله، با ساختاری شبیه به درخت که هر کندو به کندوهای مجاورش متصل است، باید کوتاه‌ترین مسافت بین دو کندو مشخص شده را پیدا کنید.
