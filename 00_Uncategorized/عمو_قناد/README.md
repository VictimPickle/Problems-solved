# عمو_قناد

## English Summary
Given a set of pastries with different cooking times and desired cooking times, the task is to find the minimum total penalty incurred when removing the pastries one at a time within a chosen minute. The penalty for each pastry is the absolute difference between its actual cooking time and desired cooking time.

## Persian Summary / خلاصه فارسی
در این مسئله، باید با توجه به زمان‌های پخت مختلف شیرینی‌ها و زمان‌های مورد نظر برای پخت هر کدام، کمترین مجموع جریمه ممکن را محاسبه کنید که در هنگام خارج کردن شیرینی‌ها یکی یکی در یک دقیقه انتخاب شده بوجود می‌آید. جریمه هر شیرینی برابر با فاصله مطلق بین زمان واقعی پخت آن و زمان مورد نظر برای پخت آن است.
