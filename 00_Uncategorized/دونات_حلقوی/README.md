# دونات_حلقوی

## English Summary
Given a network of donut shops with direct and indirect connections, the task is to find the shortest path from the shop where Chakhuf is located to the nearest circular shop. The input consists of the number of shops, the shop where Chakhuf is located, the total number of shops, and the connections between the shops. The output should be the shortest path from Chakhuf's shop to a circular shop.

## Persian Summary / خلاصه فارسی
در این مسئله، یک شبکه از دونات فروشی‌ها با ارتباطات مستقیم و غیرمستقیم داده شده است و وظیفه این است که کوتاه‌ترین مسیر از دونات فروشی که چخوف در آن قرار دارد به نزدیک‌ترین دونات فروشی حلقوی را پیدا کند. ورودی شامل تعداد دونات فروشی‌ها، شماره دوناتی که چخوف در آن است، تعداد کل دونات فروشی‌ها و ارتباطات بین آن‌ها است. خروجی باید کوتاه‌ترین مسیر از دوناتی که چخوف در آن است به یک دونات فروشی حلقوی را نشان دهد.
