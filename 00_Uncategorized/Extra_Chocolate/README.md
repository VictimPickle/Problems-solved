# Extra_Chocolate

## English Summary
Manika needs to distribute n chocolates into b boxes such that each box has an equal number of chocolates. She must bake additional chocolates to achieve this distribution. The task is to determine the minimum number of additional chocolates she needs to bake.

## Persian Summary / خلاصه فارسی
مانیکا باید n شکلات را در b جعبه قرار دهد به طوری که هر جعبه تعداد یکسانی شکلات داشته باشد. او باید تعدادی شکلات اضافی بپزد تا به این توزیع برسد. وظیفه این است که تعداد کمترین شکلات اضافی را که باید بپزد، محاسبه کند.
