# BFS_بلدی_یا_نه؟

## English Summary
Given a tree with $n$ nodes, the task is to determine if a given sequence represents a BFS traversal starting from node 1. The input consists of the number of nodes $n$, the tree edges, and the sequence. The output should be "Yes" if the sequence is a valid BFS traversal starting from node 1, and "No" otherwise.

## Persian Summary / خلاصه فارسی
با داشتن یک درخت با $n$ راس، وظیفه این است که تشخیص دهید آیا یک دنباله داده شده نمایانگر یک پیمایش BFS با شروع از راس 1 است یا خیر. ورودی شامل تعداد راس‌ها $n$، یال‌های درخت و دنباله می‌باشد. خروجی باید "Yes" باشد اگر دنباله یک پیمایش BFS معتبر با شروع از راس 1 باشد و در غیر این صورت "No".
