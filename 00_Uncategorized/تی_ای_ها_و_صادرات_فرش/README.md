# تی_ای_ها_و_صادرات_فرش

## English Summary
The problem involves creating a specific carpet pattern using a recursive algorithm. The carpet pattern is constructed by drawing squares and connecting their edges in a specific manner. The input is an integer n representing the size of the carpet pattern. The output should be the carpet pattern displayed using '#' for square edges and '.' for empty spaces.

## Persian Summary / خلاصه فارسی
این مسئله شامل ساخت یک الگوی فرش خاص با استفاده از یک الگوریتم بازگشتی است. الگوی فرش با رسم مربع‌ها و اتصال لبه‌های آن‌ها به یک شیوه خاص ساخته می‌شود. ورودی یک عدد صحیح n که اندازه الگوی فرش را نشان می‌دهد است. خروجی باید الگوی فرش با استفاده از '#' برای لبه‌های مربع و '.' برای فضاهای خالی نمایش داده شود.
