# زبان_کولوبیمینیونی

## English Summary
The problem involves checking if a given word belongs to a language generated by a specified grammar. The input consists of the grammar rules followed by the word to be tested. The task is to determine if the word is part of the language generated by the grammar and output "yes" or "no" accordingly.

## Persian Summary / خلاصه فارسی
در این مسئله باید بررسی کنید که آیا یک کلمه مشخص به یک زبانی که توسط یک گرامر مشخص تولید می‌شود تعلق دارد یا خیر. ورودی شامل قوانین گرامر و سپس کلمه‌ای است که بیاید بررسی شود. وظیفه شما این است که تشخیص دهید که آیا کلمه مورد نظر در زبان تولید شده توسط گرامر است یا خیر و به تبع آن "yes" یا "no" را چاپ کنید.
