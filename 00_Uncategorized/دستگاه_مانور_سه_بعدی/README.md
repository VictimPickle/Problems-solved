# دستگاه_مانور_سه_بعدی

## English Summary
Given a table of dimensions $n*n$ with blocks of size $1*1$, the task is to move all blocks in the specified directions (Right, Up, Down, Left) as per the given sequence and provide the final table configuration.

## Persian Summary / خلاصه فارسی
با داشتن یک جدول با ابعاد $n*n$ و بلوک‌هایی به اندازه $1*1$، وظیفه این است که تمام بلوک‌ها را به ترتیب جهت‌های مشخص شده (راست، بالا، پایین، چپ) حرکت داده و تنظیم نهایی جدول را ارائه دهید.
