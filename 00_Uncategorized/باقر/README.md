# باقر

## English Summary
The problem involves implementing a bubble sort algorithm on a given sequence of numbers and constructing a graph based on the swaps made during the sorting process. The task is to find the size of the maximum independent set in the resulting graph.

## Persian Summary / خلاصه فارسی
در این مسئله باید الگوریتم مرتب‌سازی حبابی را بر روی یک دنباله اعداد اجرا کرده و یک گراف بر اساس تعویض‌های انجام شده در فرآیند مرتب‌سازی ساخته شود. وظیفه این است که اندازهٔ بیشینهٔ مجموعهٔ مستقل در گراف حاصل را بیابید.
