# twoArray

## English Summary
Given two sorted arrays of natural numbers with lengths $n$ and $m$, for each element in the first array, print the largest element in the second array that is smaller than it. You need to output $n$ numbers representing the solution for each element in the first array.

## Persian Summary / خلاصه فارسی
در این مسئله دو آرایه‌ی مرتب شده از اعداد طبیعی به طول $n$ و $m$ داده می‌شود. برای هر عضو از آرایه‌ی اول، بزرگترین عضو آرایه‌ی دوم که از آن کوچکتر است را چاپ کنید. باید $n$ عدد را چاپ کنید که برای هر عضو از آرایه‌ی اول جواب مساله را نشان می‌دهد.
