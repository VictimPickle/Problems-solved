# Down2Zero

## English Summary
Given two positive integers $n$ and $k$, design a program to find the minimum number of steps required to reach from number $n$ to $0$ using the following steps:
1. *Step 1*: Subtract one from the current number.
2. *Step 2*: If the current number is divisible by $k$, then divide it by $k$.

## Persian Summary / خلاصه فارسی
با دو عدد مثبت $n$ و $k$ داده شده، برنامه‌ای طراحی کنید تا کمترین تعداد گام‌های مورد نیاز برای رسیدن از عدد $n$ به $0$ را با استفاده از گام‌های ذکر شده پیدا کند.
