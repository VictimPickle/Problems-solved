# نقش_و_نگار_طومار_اژدها

## English Summary
The problem involves reconstructing dragon patterns on a square paper grid by repeatedly dividing and coloring white squares based on a given model. The input specifies the dimensions of the model and the number of stages, along with the initial grid pattern. The output requires printing the final grid pattern after applying the coloring rules for the specified number of stages.

## Persian Summary / خلاصه فارسی
این مسئله شامل بازسازی الگوهای اژدها بر روی یک توره مربعی با تقسیم و رنگ‌آمیزی تکراری خانه‌های سفید بر اساس یک مدل داده شده است. ورودی ابعاد مدل و تعداد مراحل را همراه با الگوی اولیه توره را مشخص می‌کند. خروجی نیازمند چاپ الگوی نهایی توره پس از اعمال قوانین رنگ‌آمیزی برای تعداد مشخصی از مراحل است.
