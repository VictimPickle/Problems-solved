# اجداد_گره

## English Summary
Given a directed acyclic graph (DAG) with $n$ nodes and $m$ edges, the task is to find and output the ancestors of each node in ascending order.

## Persian Summary / خلاصه فارسی
در این مسئله، یک گراف جهتدار بدون دور (DAG) با $n$ گره و $m$ یال داده شده است و باید اجداد هر گره را به ترتیب صعودی پیدا کرده و خروجی دهید.
