# پیتیکو

## English Summary
Given a chessboard of size n by n, with the current position of a knight and its destination provided, calculate the minimum number of moves required to reach the destination square. The possible moves for the knight are shown in the image. The use of stack and either DFS or BFS algorithm is mandatory in this problem.

## Persian Summary / خلاصه فارسی
در این مسئله، با یک صفحه شطرنج n در n و موقعیت فعلی و مقصد یک مهره اسب مواجه هستیم. باید حداقل تعداد حرکت‌های لازم برای رسیدن مهره اسب به مقصد را محاسبه کنیم. حرکات ممکن برای اسب نشان داده شده‌اند و استفاده از stack و یکی از الگوریتم‌های DFS یا BFS در این مسئله الزامی است.
