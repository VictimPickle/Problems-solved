# گور_چه_خوفی؟!

## English Summary
The problem involves a graph representing the body of a fictional animal called "Gourkhor." Each vertex in the graph is either black or white, and the Gourkhor can change the color of all vertices in a connected component to the color of a selected vertex. The goal is to find the minimum number of operations needed to make all vertices of the Gourkhor the same color.

## Persian Summary / خلاصه فارسی
در این مسئله یک گراف به عنوان بدن حیوان خیالی به نام "گورخر" وجود دارد. هر راس در گراف به رنگ سیاه یا سفید است و گورخر می‌تواند رنگ تمام رئوس یک مولفه همبندی را به رنگ یکی از رئوس انتخابی تغییر دهد. هدف این است که کمترین تعداد عملیات مورد نیاز برای هم‌رنگ شدن تمام رئوس گورخر را پیدا کنیم.
