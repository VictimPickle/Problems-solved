# آرایه_یکنواخت

## English Summary
Given an array of integers, the task is to make all elements of the array equal by performing specific operations on pairs of adjacent elements. The operations involve updating an element based on the absolute difference between itself and its adjacent element. The goal is to determine the minimum number of operations required to make all elements equal.

## Persian Summary / خلاصه فارسی
در این مسئله، با داده شدن یک آرایه از اعداد صحیح، باید با انجام عملیات‌های خاص بر روی جفت‌های مجاور اعداد، تمامی عناصر آرایه را یکسان کنید. این عملیات شامل به‌روزرسانی یک عنصر بر اساس فاصله مطلق بین خود و عنصر مجاورش می‌باشد. هدف این است که حداقل تعداد عملیات مورد نیاز برای یکنواخت کردن تمامی عناصر را تعیین کنید.
