# فضای_سبز

## English Summary
The main character wants to go to the green space at the university after classes to relax. The university is represented as an $n \times n$ grid. The character can only move one step to the right or one step down due to limited navigation abilities. Some cells in the grid contain bugs, which the character cannot enter. The task is to calculate the number of possible paths from the starting cell to the destination cell, considering the character's movement constraints.

## Persian Summary / خلاصه فارسی
شخصیت اصلی پس از پایان کلاس‌ها می‌خواهد به فضای سبز دانشگاه برود تا استراحت کند. دانشگاه به صورت یک جدول $n \times n$ مدل‌سازی شده است. شخصیت تنها می‌تواند یک گام به راست یا یک گام به پایین حرکت کند و نمی‌تواند وارد خانه‌های دارای سوسک شود. وظیفه‌ی شما محاسبه تعداد مسیرهای ممکن از خانه‌ی شروع تا خانه‌ی مقصد با رعایت محدودیت‌های حرکت شخصیت است.
