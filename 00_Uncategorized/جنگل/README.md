# جنگل

## English Summary
Given a binary $m\times n$ matrix representing the presence of trees, where 1 indicates a tree and 0 indicates no tree, the task is to find the minimum number of days needed to disconnect the matrix into multiple disconnected forests by cutting down trees.

## Persian Summary / خلاصه فارسی
در این مسئله، با داده شدن یک ماتریس باینری $m\times n$ که نمایانگر وجود یا عدم وجود درخت‌ها است، باید حداقل تعداد روزهای مورد نیاز برای جدا کردن ماتریس به چند جنگل غیرمتصل با قطع درختان را محاسبه کنید.
