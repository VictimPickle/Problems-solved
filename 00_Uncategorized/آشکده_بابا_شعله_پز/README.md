# آشکده_بابا_شعله_پز

## English Summary
The problem involves setting up a system for a famous noodle shop where customers can donate their reserved noodles to others in need. The system should match donation requests with available donations immediately. Each request or donation should result in a printed confirmation.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک سیستم برای یک آشکده معروف ایجاد کرد که مشتریان می‌توانند آش رزرو شده خود را به دیگران درخواست کنند. سیستم باید درخواست‌ها و اهداها را بلافاصله با یکدیگر مطابقت دهد و هر درخواست یا اهدا با یک تأیید چاپ شود.
