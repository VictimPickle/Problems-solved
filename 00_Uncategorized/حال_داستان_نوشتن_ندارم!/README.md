# حال_داستان_نوشتن_ندارم!

## English Summary
Given an array of integers, the task is to find the largest k-heap that can be constructed from the array. A k-heap is defined as a sequence of numbers in the form $1, 2, ..., k-1, k, k-1, ..., 2, 1$. The operations allowed are decrementing any element by one or removing the first or last element of the array.

## Persian Summary / خلاصه فارسی
با داده شدن یک آرایه از اعداد صحیح، وظیفه شما این است که بزرگترین k-هرمی را که می‌توان از آرایه ساخت، پیدا کنید. یک k-هرم به عنوان یک دنباله اعداد به شکل $1, 2, ..., k-1, k, k-1, ..., 2, 1$ تعریف می‌شود. عملیات‌های مجاز شامل کاهش یک عنصر به مقدار یک و یا حذف کردن اولین یا آخرین عنصر آرایه هستند.
