# خفن‌ترین_هواداران

## English Summary
Given a row of football fans represented by '0' and '1', where each adjacent group of fans sitting together supports the same team, we need to find the size of the largest group of fans supporting team '0'.

## Persian Summary / خلاصه فارسی
با داشتن یک ردیف از هواداران فوتبال که با '0' و '1' نمایش داده شده‌اند و هر گروه متوالی از هواداران که کنار هم نشسته‌اند تیم یکسانی را حمایت می‌کنند، ما باید اندازه بزرگترین گروه هواداران حامی تیم '0' را پیدا کنیم.
