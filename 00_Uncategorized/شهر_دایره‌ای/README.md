# شهر_دایره‌ای

## English Summary
Given a circular city with a main square and two types of streets (direct and circular), the task is to determine if it is possible to travel from any intersection to another by following the specified directions on the streets. The streets are one-way, and the directions may vary. The input includes the number of direct and circular streets, along with the directions of each street. The output should be "YES" if it is possible to travel between any two intersections, and "NO" otherwise.

## Persian Summary / خلاصه فارسی
در این مسئله باید بررسی شود آیا می‌توان از هر تقاطعی به هر تقاطع دیگری در شهر دایره‌ای مسافت طی کرد یا خیر. خیابان‌ها یک‌طرفه هستند و جهت آن‌ها ممکن است متفاوت باشد. ورودی شامل تعداد خیابان‌های مستقیم و دایره‌ای و جهت هر خیابان است. خروجی باید "YES" اگر امکان سفر بین هر دو تقاطع وجود داشته باشد و در غیر اینصورت "NO" باشد.
