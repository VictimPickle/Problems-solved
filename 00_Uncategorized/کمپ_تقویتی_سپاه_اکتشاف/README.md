# کمپ_تقویتی_سپاه_اکتشاف

## English Summary
The problem involves determining whether a given $n*n$ table satisfies the rules of a Sudoku puzzle. Each row, column, and $\sqrt{n} * \sqrt{n}$ subgrid must contain the numbers $1$ to $n$ exactly once. The task is to write a program to check if the input table is a valid Sudoku solution.

## Persian Summary / خلاصه فارسی
در این مسئله باید تشخیص داده شود که آیا یک جدول $n*n$ داده شده، قوانین یک پازل سودوکو را رعایت می‌کند یا خیر. هر سطر، ستون و زیرجدول $\sqrt{n} * \sqrt{n}$ باید شامل اعداد $1$ تا $n$ دقیقا یک بار باشند. وظیفه این است که برنامه‌ای بنویسید تا بررسی کند آیا جدول ورودی یک جواب صحیح برای سودوکو است یا خیر.
