# graphCenter

## English Summary
The problem requires writing a program that reads a weighted graph as input and outputs the number of the central vertex. The central vertex is the one with the maximum distance to the other vertices being minimum.

## Persian Summary / خلاصه فارسی
در این مسئله باید برنامه‌ای نوشت که یک گراف وزن‌دار را از ورودی خوانده و شماره‌ی راس مرکز را خروجی دهد. راس مرکز، راسی است که بیشینه فاصله‌اش از باقی رئوس کمینه باشد.
