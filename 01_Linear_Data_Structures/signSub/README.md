# signSub

## English Summary
The problem requires writing a program to read two signed integers from input and output their difference without using the '-' operator directly.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک برنامه بنویسید که دو عدد صحیح علامت‌دار را از ورودی بخواند و اختلاف آن‌ها را بدون استفاده مستقیم از عملگر '-' در خروجی چاپ کند.
