# stringMatch

## English Summary
Given two strings $A$ and $B$, the task is to find the maximum value of $i$ such that $B^i$ is a substring of $A$. The program should output the maximum value of $i$ and the starting index of the substring $B^i$ in string $A$.

## Persian Summary / خلاصه فارسی
با داشتن دو رشته $A$ و $B$، وظیفه برنامه این است که بزرگترین مقدار $i$ را پیدا کند به‌طوری که $B^i$ یک زیررشته از $A$ باشد. برنامه باید مقدار بیشینه $i$ و اندیس شروع زیررشته $B^i$ در رشته $A$ را خروجی دهد.
