# Maze

## English Summary
The problem involves generating a spiral square matrix of size n by n filled with 0s and 1s using a recursive function and printing the resulting matrix.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک ماتریس مربعی مارپیچ به ابعاد n در n حاوی اعداد 0 و 1 با استفاده از یک تابع بازگشتی ایجاد کرده و ماتریس حاصل را چاپ کنید.
