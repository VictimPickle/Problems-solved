# خسته_و_تنها

## English Summary
The problem involves compressing or expanding words based on specific rules. Given a word, if there are consecutive identical characters, they should be replaced with the character followed by the count of repetitions. The case sensitivity of characters should be considered. The task is to process a list of words and perform compression or expansion based on a given number for each word.

## Persian Summary / خلاصه فارسی
در این مسئله باید کلمات را بر اساس قوانین خاصی فشرده یا گسترده کرد. با توجه به یک کلمه، اگر حروف متوالی یکسان وجود داشته باشند، باید آن‌ها را با حرف تکرار شده به همراه تعداد تکرار جایگزین کرد. حساسیت به بزرگی و کوچکی حروف باید در نظر گرفته شود. وظیفه این است که یک لیست از کلمات را پردازش کرده و بر اساس یک عدد داده شده برای هر کلمه، فشرده یا گسترده سازی را انجام دهید.
