# رشته_رمزی

## English Summary
Given a set of sentences and corresponding encoded strings, the task is to determine if the encoded string can represent the sentence by mapping each letter in the encoded string to at least one letter in the sentence.

## Persian Summary / خلاصه فارسی
در این مسئله، باید بررسی شود که آیا رشته رمزی می‌تواند یک جمله را نمایندگی کند یا خیر، با تطابق هر حرف در رشته رمزی با حداقل یک حرف در جمله.
