# مجموع_زیرآرایه

## English Summary
Given an array of integers, the task is to determine if there exists at least one subarray with a sum of zero. Print 1 if such a subarray exists, otherwise print 0. It is recommended to use the STL map for implementing this problem.

## Persian Summary / خلاصه فارسی
در این مسئله، یک آرایه از اعداد صحیح داده می‌شود و باید بررسی شود که آیا حداقل یک زیرآرایه وجود دارد که مجموع اعضای آن برابر با صفر باشد یا خیر. در صورت وجود چنین زیرآرایه‌ای، عدد ۱ و در غیر این صورت عدد ۰ چاپ می‌شود. برای پیاده‌سازی این مسئله، استفاده از STL مپ توصیه می‌شود.
