# بانک_اطلاعات_فوق_سری_مارلی

## English Summary
The problem involves finding the least common multiple (LCM) of two given strings. The task is to determine the LCM of the strings formed by concatenating multiple occurrences of the input strings. The goal is to find the LCM for a given number of queries.

## Persian Summary / خلاصه فارسی
در این مسئله باید ک.م.م دو رشته ورودی را پیدا کنید. وظیفه شما این است که ک.م.م رشته‌ها را که از ترکیب چندین تکرار از رشته‌های ورودی ساخته شده‌اند، تعیین کنید. هدف این است که بقدری کد امنیتی را برای تعدادی درخواست مشخص شده پیدا کنید.
