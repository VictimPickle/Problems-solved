# ternaryCode

## English Summary
The problem involves decoding a given string based on the Ternary numeric notation used in Berland. The input string consists of dots and dashes representing the ternary digits 0, 1, and 2. The task is to translate this notation into the corresponding ternary numbers and output the decoded value.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک رشته ورودی را بر اساس notation سه‌گانه معروف در Berland که از نقطه و خط استفاده می‌کند decode کنید. رشته ورودی شامل نقطه و خط‌ها که اعداد سه‌گانه 0، 1 و 2 را نمایندگی می‌کنند است. وظیفه شما ترجمه این notation به اعداد سه‌گانه متناظر و نمایش مقدار decode شده است.
