# اعصار_قدیم

## English Summary
Given two strings $a$ and $b$, the task is to find the smallest merged word that can be formed by merging two words if and only if there exists a word that contains both words as substrings. The output should be the smallest merged word based on the specified condition.

## Persian Summary / خلاصه فارسی
با داشتن دو رشته $a$ و $b$، وظیفه‌ی مسئله این است که کوچک‌ترین کلمه‌ای که می‌توان از ادغام دو کلمه به دست آورد را پیدا کند، در صورتی که تنها اگر یک کلمه وجود داشته باشد که هر دو کلمه را به عنوان زیررشته‌ها داشته باشد. خروجی باید کوچک‌ترین کلمه‌ی ادغام شده بر اساس شرط مشخص شده باشد.
