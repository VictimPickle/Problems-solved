# Testcase_Dilemma

## English Summary
Given a set of test cases, each test case requires finding the k-th element in a sequence where numbers divisible by 3 or ending with 3 are excluded. The task is to output the k-th element in the sequence for each test case.

## Persian Summary / خلاصه فارسی
با داده شدن مجموعه‌ای از تست کیس‌ها، هر تست کیس نیازمند پیدا کردن عنصر k ام در یک دنباله است که اعداد قابل قسمت بر 3 یا به پایان رسیدن با 3 محروم شده‌اند. وظیفه این است که برای هر تست کیس عنصر k ام در دنباله را خروجی دهد.
