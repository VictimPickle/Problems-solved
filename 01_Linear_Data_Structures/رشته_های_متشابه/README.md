# رشته_های_متشابه

## English Summary
Given two strings of equal length in a special language, we need to determine if they are similar to each other based on certain rules. The rules state that two strings are similar if they are either equal or can be split into two equal halves where the corresponding halves are similar to each other when swapped. We are asked to write a program to determine if the input strings are similar or not.

## Persian Summary / خلاصه فارسی
با داشتن دو رشته با طول یکسان در یک زبان خاص، باید تشخیص دهیم که آیا آنها به یکدیگر شبیه هستند یا خیر بر اساس قوانین خاص. این قوانین بیان می‌کنند که دو رشته زمانی به یکدیگر شبیه هستند که یا برابر باشند یا بتوانند به دو نیمه مساوی تقسیم شوند به طوری که نیمه‌های متناظر با یکدیگر شبیه هم باشند هنگامی که جابه‌جا شوند. ما باید یک برنامه بنویسیم تا تشخیص دهد که آیا رشته‌های ورودی شبیه یکدیگر هستند یا خیر.
