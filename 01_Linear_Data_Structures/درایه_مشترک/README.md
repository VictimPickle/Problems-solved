# درایه_مشترک

## English Summary
Given a $4 \times 4$ matrix, the task is to find and print all elements that are maximum in both their row and column.

## Persian Summary / خلاصه فارسی
با داشتن یک ماتریس $4 \times 4$، وظیفه این است که تمام عناصری که در هم سطر و هم ستون خود بیشینه هستند را پیدا کرده و چاپ کند.
