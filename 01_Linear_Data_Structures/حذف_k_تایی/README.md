# حذف_k_تایی

## English Summary
Given an array of numbers, the task is to remove k elements that are smaller than their next element in the array or become smaller after removing subsequent elements, and return the final array.

## Persian Summary / خلاصه فارسی
در این مسئله، یک آرایه از اعداد داده شده است و باید k عضو که از عضو بعدی خود در آرایه کوچکتر هستند یا پس از حذف اعضای بعدی کوچکتر می‌شوند را حذف کرده و آرایه‌ی نهایی را برگردانید.
