# نجات_انسانیت

## English Summary
The problem involves determining whether a given string is a palindrome or can be turned into a palindrome by changing exactly one character. The task is to help a character named Arne in deciphering codes to potentially save humanity.

## Persian Summary / خلاصه فارسی
در این مسئله باید تشخیص داد که آیا یک رشته داده شده یک پالیندروم است یا با تغییر یک حرف به پالیندروم تبدیل می‌شود. وظیفه شما کمک به شخصی به نام آرن در رمزگشایی کدها برای نجات احتمالی انسانیت است.
