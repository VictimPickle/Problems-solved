# شمارش_وارونه‌ها

## English Summary
The problem involves counting the number of inversions in an array. An inversion occurs when two elements in the array have a descending order of values based on their indices. The task is to determine the total count of inversions in the given array.

## Persian Summary / خلاصه فارسی
در این مسئله باید تعداد وارونگی‌ها در یک آرایه را شمارش کنیم. وارونگی زمانی رخ می‌دهد که دو عنصر در آرایه با ترتیب نزولی مقادیر بر اساس اندیس‌هایشان قرار دارند. وظیفه ما تعیین تعداد کل وارونگی‌ها در آرایه داده شده است.
