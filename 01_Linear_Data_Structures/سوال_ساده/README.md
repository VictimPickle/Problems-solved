# سوال_ساده

## English Summary
Given a list of $n$ integers, the task is to calculate the sum, average, maximum, and minimum of the numbers with precision up to 6 decimal places.

## Persian Summary / خلاصه فارسی
با داشتن یک لیست از $n$ اعداد صحیح، وظیفه‌ی شما این است که مجموع، میانگین، بیشینه و کمینه‌ی اعداد را با دقت تا 6 رقم اعشار محاسبه کنید.
