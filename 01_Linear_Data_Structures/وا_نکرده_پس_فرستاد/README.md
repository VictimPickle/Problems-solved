# وا_نکرده_پس_فرستاد

## English Summary
Given a text editor with specific operations, the task is to perform a series of operations and print the final text after applying those operations. The operations include moving the cursor left, right, inserting, deleting characters, and replacing characters in the text.

## Persian Summary / خلاصه فارسی
در این مسئله، با یک ویرایشگر متن و عملیات‌های خاص آن روبرو هستیم. وظیفه‌ی ما انجام یک سری عملیات و چاپ متن نهایی پس از اعمال این عملیات است. این عملیات شامل حرکت اشاره‌گر به چپ و راست، درج و حذف کاراکترها، و جایگزینی کاراکترها در متن می‌باشد.
