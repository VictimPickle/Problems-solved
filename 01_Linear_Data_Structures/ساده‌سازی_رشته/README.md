# ساده‌سازی_رشته

## English Summary
The problem involves simplifying a given string by removing consecutive pairs of different characters until no such pairs exist.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک رشته داده شده را ساده‌سازی کرده و حروف متوالی متفاوت را حذف کرد تا هیچ جفتی از حروف متفاوت باقی نماند.
