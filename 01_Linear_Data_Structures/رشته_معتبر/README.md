# رشته_معتبر

## English Summary
Given a string of length n, the task is to find and print the longest substring that represents a valid identifier name. An identifier name is considered valid if it consists of a combination of uppercase letters (A-Z), lowercase letters (a-z), digits (0-9), or underscore (_) and does not start with a number.

## Persian Summary / خلاصه فارسی
در این مسئله با داشتن یک رشته به طول n، باید طولانی‌ترین زیر رشته‌ای که یک اسم شناسه معتبر را نمایندگی می‌کند را پیدا و چاپ کنید. یک اسم شناسه به عنوان معتبر در نظر گرفته می‌شود اگر از ترکیبی از حروف بزرگ (A-Z)، حروف کوچک (a-z)، ارقام (0-9) یا زیرخط (_) تشکیل شده باشد و با یک عدد شروع نشود.
