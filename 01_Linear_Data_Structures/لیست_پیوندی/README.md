# لیست_پیوندی

## English Summary
Given a singly linked list of size $n$, rearrange the adjacent pairs of nodes and print the final list. The nodes themselves should be swapped, not just their values. The program should have a time complexity of $O(n)$.

## Persian Summary / خلاصه فارسی
در این مسئله، یک لیست پیوندی یک طرفه به اندازه $n$ داده شده است. باید جفت‌های مجاور node ها را جابه‌جا کرده و لیست نهایی را چاپ کرد. در این جابه‌جایی، خود node ها باید عوض شوند و صرفا مقادیر آن‌ها نه. برنامه باید دارای پیچیدگی زمانی $O(n)$ باشد.
