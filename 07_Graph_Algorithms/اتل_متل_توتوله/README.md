# اتل_متل_توتوله

## English Summary
The problem involves simulating a game called "Atel Metel Tutule" where players move from one foot to another while reciting a poem. The goal is to write a program that outputs the sequence of foot movements in each round of the game and determines the winner. The game continues until only one player has a foot left.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک برنامه بنویسید که بازی "اتل متل توتوله" را شبیه‌سازی کند. در این بازی، بازیکنان با خواندن یک شعر از یک پا به پای دیگر حرکت می‌کنند و باید ترتیب حرکت پاها در هر مرحله و برنده نهایی را تعیین کند. بازی تا زمانی ادامه می‌یابد که تنها یک بازیکن پا داشته باشد.
