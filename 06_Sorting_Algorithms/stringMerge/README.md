# stringMerge

## English Summary
The problem requires writing a program that takes two strings of lengths $n$ and $m$, sorts each string separately, and then merges the two sorted strings without using pre-built sorting functions like Sort.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک برنامه نوشت که دو رشته به طول‌های $n$ و $m$ را گرفته، هر رشته را به طور جداگانه مرتب کند و سپس این دو رشته مرتب شده را بدون استفاده از توابع مرتب‌سازی از پیش ساخته شده، ادغام کند.
