# حمله‌ی_مرتب

## English Summary
The problem involves merging two sorted arrays efficiently in $O(n + m)$ time complexity, where $n$ and $m$ are the lengths of the arrays. The goal is to combine the arrays into a single sorted array without violating the sorting order.

## Persian Summary / خلاصه فارسی
در این مسئله باید دو آرایه مرتب شده را به صورت کارآمد در زمان $O(n + m)$ که $n$ و $m$ طول آرایه‌ها هستند، ترکیب کرده و یک آرایه مرتب ترکیبی از آن‌ها را بدست آورید بدون اینکه ترتیب مرتب سازی نقض شود.
