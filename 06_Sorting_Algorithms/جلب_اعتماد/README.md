# جلب_اعتماد

## English Summary
The problem involves decoding a permutation of numbers using a given merge sort algorithm. The task is to determine the original permutation based on the sorted output produced by the algorithm.

## Persian Summary / خلاصه فارسی
در این مسئله باید یک جایگشت از اعداد را بازگشایی کرده و جایگشت اصلی را بر اساس خروجی مرتب‌سازی تولید شده توسط الگوریتم مشخص شده تعیین کنید.
