# پیر_هرات_و_شطرنج

## English Summary
The problem involves a chessboard designed by an inventor named Pir Harat, where some squares contain chess pieces and others are empty. Pir Harat asks for help in determining the number of chess pieces in specified rectangular subregions of the board.

## Persian Summary / خلاصه فارسی
در این مسئله، مخترعی به نام پیر هرات یک صفحه شطرنج طراحی کرده است که برخی از خانه‌های آن مهره دارند و برخی دیگر خالی هستند. پیر هرات از شما در تعیین تعداد مهره‌ها در زیرمستطیل‌های مشخص شده روی صفحه کمک می‌خواهد.
