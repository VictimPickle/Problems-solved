# قادر

## English Summary
Given a set of cities and the cost of traveling between each pair of cities, the problem is to find the minimum cost to visit all cities exactly once and return to the starting city. This is known as the Traveling Salesman Problem (TSP).

## Persian Summary / خلاصه فارسی
با داشتن مجموعه‌ای از شهرها و هزینه سفر بین هر زوج شهر، مسئله این است که هزینه کمینه برای بازدید از تمام شهرها دقیقا یک بار و بازگشت به شهر شروع را پیدا کنیم. این مسئله به عنوان مسئله فروشنده دورهگرد (TSP) شناخته می‌شود.
