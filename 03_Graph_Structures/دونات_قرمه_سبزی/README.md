# دونات_قرمه_سبزی

## English Summary
The problem involves finding the minimum number of students to whom free "Ghormeh Sabzi" stew should be given in order to attract all other students based on their relationships in a university setting.

## Persian Summary / خلاصه فارسی
در این مسئله باید حداقل تعداد دانشجویانی را که باید به آنها قرمه سبزی رایگان داده شود تا سایر دانشجویان نیز جذب شوند بر اساس روابطشان در یک محیط دانشگاهی پیدا کنید.
